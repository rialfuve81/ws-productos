🛒 Proyecto ws-productos – Spring Boot + MySQL

Este proyecto es una API REST desarrollada en Spring Boot, que implemento un CRUD a la entidad productos. La aplicación se ejecuta junto con una base de datos MySQL.

🚀 Tecnologías utilizadas

Java 17

Spring Boot

Maven

MySQL 8

📦 Requisitos previos

Antes de ejecutar el proyecto asegúrate de tener instalado:

Java 17

Maven
